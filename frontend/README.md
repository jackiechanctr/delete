# Frontend

Next.js 15 dashboard for the Unified Wealth Intelligence Platform.

## Stack

Next.js, TypeScript, TailwindCSS, Framer Motion, React Query, Zustand, Recharts, Socket.IO client.

## Run

```bash
npm run dev
```

Set `NEXT_PUBLIC_API_URL` in `.env.local` (default: http://localhost:3001).
